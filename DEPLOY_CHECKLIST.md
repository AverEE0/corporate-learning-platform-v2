# ✅ Чеклист развертывания на VPS

## Перед развертыванием

- [ ] VPS сервер готов (Ubuntu 20.04+ / Debian 11+)
- [ ] Docker установлен
- [ ] Docker Compose установлен
- [ ] Firewall настроен (порты 22, 80, 443 открыты)
- [ ] Файлы проекта загружены на сервер

## Настройка

- [ ] Файл `.env` создан из `.env.production.example`
- [ ] `DATABASE_URL` настроен (внешняя БД или в Docker)
- [ ] `JWT_SECRET` установлен (случайная строка минимум 32 символа)
- [ ] `NEXT_PUBLIC_APP_URL` установлен (IP или домен)
- [ ] Все пароли изменены на безопасные

## Развертывание

- [ ] Директории созданы: `uploads`, `logs`, `ssl`
- [ ] Права доступа установлены: `chmod 755 uploads logs`
- [ ] Docker образы собраны: `docker-compose build`
- [ ] Контейнеры запущены: `docker-compose up -d`
- [ ] База данных инициализирована
- [ ] Health check проходит: `curl http://localhost/api/health`

## После развертывания

- [ ] Приложение доступно по IP/домену
- [ ] Страница входа открывается
- [ ] Регистрация работает
- [ ] SSL сертификат установлен (если используется домен)
- [ ] Резервное копирование настроено
- [ ] Мониторинг настроен

## Безопасность

- [ ] Все пароли изменены
- [ ] JWT_SECRET уникальный и длинный
- [ ] Firewall активен
- [ ] Fail2ban настроен (опционально)
- [ ] Регулярные обновления системы запланированы

## Готово! ✅

Если все пункты отмечены - проект готов к работе!
